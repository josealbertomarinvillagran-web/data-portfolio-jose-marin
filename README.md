# Análisis de Ventas - Power BI

## Descripción del proyecto
Este proyecto tiene como objetivo analizar el comportamiento de ventas para identificar tendencias, oportunidades de crecimiento y patrones de consumo.

## Problema de negocio
La empresa necesita comprender mejor sus ventas para optimizar la toma de decisiones comerciales.

## Herramientas utilizadas
- Power BI
- Excel
- DAX

## Proceso
- Limpieza de datos
- Modelado de datos
- Creación de medidas DAX
- Visualización de información

## Resultados / Insights
- Identificación de tendencias de ventas
- Análisis de comportamiento de clientes
- Detección de oportunidades de mejora comercial
